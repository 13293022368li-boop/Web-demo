About This Project

A modern movie & anime information library built with pure HTML, CSS, and JavaScript.

Project Overview

This website was created as part of a Web Development assignment. It showcases a dynamic movie and anime library with features such as filtering, sorting, searching, and detailed information pages.

Technologies Used

  • HTML5 for structure
  • CSS3 for design and animations
  • JavaScript (MVC-based) for dynamic functionality

Features

  • Browse movies & anime
  • Search by title
  • Sort by rating / year
  • Genre filtering
  • Full detail pages with posters & trailers
